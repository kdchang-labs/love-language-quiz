# Love Language Quiz

A pure frontend static website deployable to GitHub Pages. Users complete a 15-question quiz to discover how they feel loved and cared for in relationships with family, partners, and friends.

## Files

- `index.html`: Page structure
- `styles.css`: Responsive styles
- `script.js`: Quiz questions, scoring, and result rendering
- `assets/relationship-map.svg`: Site logo

## Local Preview

Open `index.html` directly in a browser, or start a simple server in this folder:

```bash
npx serve .

python3 -m http.server 8080
```

Then visit:

```text
http://localhost:8080
```
